# Skill: Find-Skills 🦞🔍

Esta habilidade simples, mas poderosa, permite que o agente descubra seu próprio arsenal de inteligência local, garantindo a **Hermeticidade** do projeto.

## Comandos Principais
1. **Listar**: `list_dir` na pasta `.agents/skills`.
2. **Descrever**: Ler o preâmbulo (frontmatter) de cada `SKILL.md` para entender as ferramentas disponíveis.
3. **Mapear**: Verificar o `skills-lock.json` para garantir que as versões batem.

## Quando Usar
- Ao iniciar uma nova tarefa e não saber qual ferramenta usar.
- Para verificar se o "Survival Kit" foi instalado corretamente no projeto.
- Quando o usuário pergunta "O que você sabe fazer neste projeto?".
