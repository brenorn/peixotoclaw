# Projeto: NR01 - Anonimato Total 🛡️

Este é um projeto satélite gerenciado pelo PeixotoClaw Engine.

## 🎯 Objetivo de Negócio
Garantir que todas as avaliações NR01 sejam completamente anônimas e possam ser submetidas coletivamente por múltiplos usuários para o mesmo setor (GHE) e ciclo.

## 🏗️ Requisitos Específicos
1.  **Remoção de Unique Constraint**: Ajustar o banco de dados para permitir múltiplas submissões anônimas.
2.  **Fluxo de Dados**: Alinhar o sistema com o "Acervo Digital NR01".
3.  **Frontend**: Criar interface para submissão em massa sem identificação de usuário.

---
## 📜 Regras de Projeto
- **Segurança**: Anonimização em repouso e em trânsito.
- **Compliance**: Regulamentações NR01 vigentes.
