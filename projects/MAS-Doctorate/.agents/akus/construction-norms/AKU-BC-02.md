# AKU-BC-02: Lean Construction & Last Planner System (LPS)

Este conhecimento define as práticas de gestão enxuta para otimização de fluxo em obras.

## 🎯 Objetivo
Habilitar o agente a agir como um facilitador de Lean Construction, focando em confiabilidade e redução de desperdícios.

## 🏛️ Lógica Operacional (LPS)
1. **Planejamento "Deve-Pode-Vai-Fez"**:
   - **Pull Planning**: Planejamento puxado a partir do marco final.
   - **Make-Ready (Look-ahead)**: Análise de restrições para 4-6 semanas à frente.
   - **Compromisso Semanal**: Apenas tarefas sem restrições entram no plano semanal.
2. **Métricas de Performance**:
   - **PPC (Porcentagem de Promessas Concluídas)**: Medida principal de confiabilidade.
   - **Causa da Variação**: Se PPC < 100%, o agente deve perguntar/identificar o motivo (ex: falta de material, clima, mão de obra).

## 🧬 Regras de Ouro
- "Variabilidade é o inimigo do fluxo."
- "Não inicie uma atividade se as restrições não estiverem removidas."
- "O Last Planner (quem executa) deve participar do planejamento."
