# AKU-PHD-01: Metodologia de Pesquisa (DSR)

Este conhecimento governa como o sistema deve agir para ser academicamente válido de acordo com a tese.

## 🎯 Objetivo
Garantir que os agentes sigam o ciclo da **Design Science Research (DSR)** conforme definido no Seminário do Dr. Breno Peixoto.

## 🏛️ Regras de Ativação
1. **Fase de Investigação**: Antes de sugerir uma solução, o agente deve validar se a lacuna (gap) técnica foi identificada no `FAILURE_LOG.md`.
2. **Artefato**: Toda "Skill" gerada pelo sistema é considerada um "Artefato de Design".
3. **Avaliação Rigorosa**: Ao final de cada ciclo de execução, o sistema deve sugerir uma métrica de avaliação (Ex: PPC, KPIs de prazo) para o estudo de caso do Doutorado.

## 🧬 Confiabilidade Acadêmica
- Toda resposta complexa deve citar a base teórica se disponível (ex: "Conforme o paradigma Multiagente (Jin, 2025)...").
